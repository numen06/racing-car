### 简介


React开发的赛车小游戏。左右躲避对面开来的车辆，遵守红绿灯规则，争取更多的里程数。

### 运行环境

- 处理器：双核及以上处理器
- 内存：4 GB及以上
- 显卡：最低要求Super VGA（800 x 600）
- 硬盘可用空间：30G（主分区，NTFS格式）
- 浏览器：chrome内核浏览器
- 外设：具有键盘


### 操作说明

键盘上下左右控制：


- ⬆️ 前进
- ⤵️ 刹车
- ⬆️⬆️ 暂时加速
- ⬅️ / ➡️ 变道
